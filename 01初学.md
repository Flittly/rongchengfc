# React + TypeScript 项目初学指南

## 项目概述
这是一个使用 Vite 构建的 React + TypeScript 项目。Vite 是一个现代化的前端构建工具，比传统的 Create React App 更快更轻量。

## 文件结构说明

```
rongchengfc/
├── public/                 # 静态资源目录
│   ├── favicon.svg        # 网站图标
│   └── icons.svg          # SVG 图标集合
├── src/                   # 源代码目录（最重要）
│   ├── assets/           # 资源文件（图片、SVG等）
│   ├── App.css           # App组件的样式
│   ├── App.tsx           # 主组件（核心文件）
│   ├── index.css         # 全局样式
│   └── main.tsx          # 应用入口文件
├── index.html            # HTML 入口文件
├── package.json          # 项目配置和依赖
├── tsconfig.json         # TypeScript 配置
├── vite.config.ts        # Vite 配置
└── eslint.config.js      # 代码规范配置
```

## 核心文件详解

### 1. `package.json` - 项目配置
```json
{
  "name": "rongchengfc",        // 项目名称
  "scripts": {
    "dev": "vite",              // 启动开发服务器
    "build": "tsc -b && vite build",  // 构建生产版本
    "lint": "eslint ."          // 代码检查
  },
  "dependencies": {
    "react": "^19.2.4",         // React 核心库
    "react-dom": "^19.2.4"      // React DOM 渲染器
  }
}
```
**作用**：管理项目依赖和运行脚本。

### 2. `index.html` - HTML 入口
```html
<div id="root"></div>           <!-- React 应用挂载点 -->
<script type="module" src="/src/main.tsx"></script>  <!-- 入口脚本 -->
```
**作用**：网页的骨架，React 应用会渲染到 `#root` 这个 div 中。

### 3. `src/main.tsx` - 应用入口
```tsx
import { createRoot } from 'react-dom/client'
import App from './App.tsx'

createRoot(document.getElementById('root')!).render(
  <App />
)
```
**作用**：
- 导入 React 和主组件
- 将 App 组件渲染到页面上
- 这是整个应用的启动点

### 4. `src/App.tsx` - 主组件
```tsx
function App() {
  const [count, setCount] = useState(0)  // 状态管理

  return (
    <div>
      <button onClick={() => setCount(count + 1)}>
        Count is {count}
      </button>
    </div>
  )
}
```
**作用**：
- 这是你修改的主要文件
- 包含组件的结构（JSX）、逻辑和状态
- 使用 `useState` 管理组件状态

### 5. `vite.config.ts` - Vite 配置
```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],  // 启用 React 支持
})
```
**作用**：配置 Vite 构建工具，一般不需要修改。

### 6. `tsconfig.json` - TypeScript 配置
**作用**：配置 TypeScript 编译选项，初学阶段不需要关注。

## 初学者学习路径

### 第一阶段：理解基础
1. **先看 `App.tsx`**：这是你主要修改的文件
2. **理解 JSX**：HTML 和 JavaScript 的混合语法
3. **学习组件**：函数组件的基本结构

### 第二阶段：核心概念
1. **状态管理**：`useState` 的使用
2. **事件处理**：`onClick` 等事件
3. **条件渲染**：`{condition && <div>...</div>}`

### 第三阶段：进阶
1. **组件通信**：Props 传递
2. **生命周期**：`useEffect` Hook
3. **类型定义**：TypeScript 接口

## 常用命令
```bash
npm run dev      # 启动开发服务器
npm run build    # 构建生产版本
npm run lint     # 代码检查
```

## 学习建议
1. **先修改 `App.tsx`**：尝试修改按钮文字、添加新元素
2. **查看浏览器**：每次保存后浏览器会自动刷新
3. **小步修改**：每次只改一点点，观察变化
4. **查阅文档**：遇到不懂的 API 查阅 React 官方文档
